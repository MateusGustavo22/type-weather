# Type Weather

Projeto React que permite ao usuário consultar o clima atual e dos próximos 5 dias de qualquer cidade.

## Tecnologias

- React
- Next
- Typescript
- Radix
- API rest

Deploy: https://type-weather-zeta.vercel.app/

API: https://open-meteo.com/
